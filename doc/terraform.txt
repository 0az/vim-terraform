COMMANDS                                                 *terraform-commands*

This command is only available if terraform is in your PATH.

                                                       *terraform-:terraform*
`:Terraform` [args]         Invoke an arbitrary terraform command.

==============================================================================
SETTINGS                                                 *terraform-settings*

                                                        *'g:terraform_align'*
`g:terraform_align`
          Allow vim-terraform to align settings automatically with Tabularize.

          Default: 0

                                                  *'g:terraform_binary_path'*
`g:terraform_binary_path`
          Allow vim-terraform to use a custom path for the terraform binary. 

          Default: terraform

                                                  *'g:terraform_fmt_on_save'*
`g:terraform_fmt_on_save`
          Allow vim-terraform to automatically format *.tf and *.tfvars files 
          with terraform fmt. You can also do this manually with the 
          `:TerraformFmt` command.
          
          Default: 0

                                                *'g:terraform_fold_sections'*
`g:terraform_fold_sections`
          Allow vim-terraform to automatically fold (hide until unfolded) 
          sections of terraform code.
          
          Default: 0
